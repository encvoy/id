## ✨ Capacidades Principales

- Inicio de sesión único (SSO)
- OpenID Connect (OIDC) y OAuth 2.0
- Control de acceso basado en roles (RBAC)
- Autoregistro de usuarios y aprovisionamiento automático
- Gestión de aplicaciones y usuarios
- Experiencia de inicio de sesión personalizable
- Registro y auditoría de eventos de seguridad

## 🔑 Métodos de Autenticación e Inicio de Sesión

Encvoy ID admite una amplia gama de métodos de autenticación,
incluidas opciones sin contraseña y resistentes al phishing.

Basado en estándares ampliamente adoptados:

- OpenID Connect (OIDC)
- OAuth 2.0

### Métodos estándar

- Nombre de usuario y contraseña
- Inicio de sesión por correo electrónico

### Proveedores sociales y externos

- Google, GitHub y otros proveedores
- Proveedores de identidad externos

### Métodos fuertes y sin contraseña

- WebAuthn / Passkeys
- mTLS (certificados de cliente)
- Contraseñas de un solo uso TOTP / HOTP

Admite autenticación multifactor (MFA) combinando múltiples factores de verificación.

Ayuda a cumplir con requisitos de seguridad y cumplimiento más altos.
Encvoy ID está diseñado para admitir una transición gradual hacia la autenticación sin contraseña.
