# Backend for SSO Encvoy ID

This project represents the backend part of the Encvoy ID Single Sign-On (SSO) system. The backend implements functionality for user management, authentication, authorization, and integration with external providers.

## Key Features

- **Authentication and Authorization**:

  - Support for various providers.
  - Role-based access control.

- **User Management**:

  - Registration, password recovery, profile management.
  - Blocking and deleting profiles.

- **Integration with External Systems**:

  - Working with external APIs for data synchronization.

- **Flexible Configuration**:
  - Support for custom rules and security policies.
  - Configurable parameters through configuration files.

## Technologies

- **Node.js**: The main runtime for executing server-side code.
- **NestJS**: A framework for building modular architectures.
- **Prisma**: ORM for database operations.
- **PostgreSQL**: Relational database.
- **Redis**: Used for caching and storing temporary data.
