# Dashboard for SSO Encvoy ID

This project represents the client-side of the Encvoy ID Single Sign-On (SSO) system. The dashboard provides a user-friendly interface for managing users, system settings, and integration with external providers.

## Key Features

- **User Management**:

  - View user list.
  - Manage profiles, roles, and access.
  - Block and delete users.

- **Application Management**:

  - View application list.
  - Manage applications, login methods, and settings.
  - View activity logs.

- **System Configuration**:

  - Manage authentication providers.
  - Configure security policies and system parameters.

- **Integration with External Systems**:

  - Connect and manage external providers.
  - Synchronize data with external APIs.

- **Analytics and Monitoring**:
  - View system usage statistics.
  - Log user actions.

## Technologies

- **React**: The main framework for building the user interface.
- **TypeScript**: Used for type safety and code reliability.
- **Redux Toolkit**: Application state management.
- **Material-UI**: Component library for creating a modern interface.
