# Widget für SSO Encvoy ID

Dieses Projekt ist ein Widget zur Integration mit dem Single Sign-On (SSO) System Encvoy ID. Das Widget bietet eine
Oberfläche für Benutzerautorisierung, Registrierung, Passwortwiederherstellung und andere zugriffsbezogene Funktionen.

## Hauptfunktionen

- **Benutzerauthentifizierung**:
  - Anmeldung mit E-Mail, Telefon oder externen Anbietern (OAuth2 usw.).
  - Unterstützung für Zwei-Faktor-Authentifizierung (2FA).
  - Unterstützung für die Protokolle webAuthn und mTLS.

- **Registrierung und Passwortwiederherstellung**:
  - Registrierung neuer Benutzer.
  - Wiederherstellung des Zugangs per E-Mail oder Telefon.

- **Anpassung**:
  - Flexible Anpassung des Erscheinungsbildes und Verhaltens des Widgets.

- **Integration**:
  - Einfache Integration in Webanwendungen.
  - Arbeitet über eine REST-API zur Interaktion mit der Server-Komponente.

## Technologien

- **React**: Das Hauptframework zum Erstellen der Benutzeroberfläche.
- **TypeScript**: Wird für Typisierung und zur Erhöhung der Zuverlässigkeit des Codes verwendet.
- **Redux Toolkit**: Verwaltung des Anwendungszustands.
- **Material-UI**: Komponentenbibliothek zur Erstellung einer modernen Oberfläche.
