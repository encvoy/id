# Widget для SSO Encvoy ID

Этот проект представляет собой виджет для интеграции с системой единого входа (SSO) Encvoy ID. Виджет предоставляет
интерфейс для аутентификации пользователей, регистрации, восстановления пароля и других функций, связанных с доступом.

## Основные возможности

- **Аутентификация пользователей**:
  - Вход с использованием email, телефона или внешних провайдеров (OAuth2 и др.).
  - Поддержка двухфакторной аутентификации (2FA).
  - Поддержка протоколов webAuthn и mTLS

- **Регистрация и восстановление пароля**:
  - Регистрация новых пользователей.
  - Восстановление доступа через email или телефон.

- **Кастомизация**:
  - Гибкая настройка внешнего вида и поведения виджета.

- **Интеграция**:
  - Лёгкая интеграция с веб-приложениями.
  - Работа через REST API для взаимодействия с серверной частью.

## Технологии

- **React**: Основной фреймворк для построения пользовательского интерфейса.
- **TypeScript**: Используется для типизации и повышения надёжности кода.
- **Redux Toolkit**: Управление состоянием приложения.
- **Material-UI**: Библиотека компонентов для создания современного интерфейса.
