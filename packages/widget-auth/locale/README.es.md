# Widget para SSO Encvoy ID

Este proyecto es un widget para integrarse con el sistema de inicio de sesión único (SSO) Encvoy ID. El widget
proporciona una interfaz para la autenticación de usuarios, el registro, la recuperación de contraseñas y otras
funciones relacionadas con el acceso.

## Características principales

- **Autenticación de usuarios**:
  - Inicio de sesión usando correo electrónico, teléfono o proveedores externos (OAuth2, etc.).
  - Soporte para autenticación de dos factores (2FA).
  - Soporte para los protocolos webAuthn y mTLS.

- **Registro y recuperación de contraseña**:
  - Registro de nuevos usuarios.
  - Recuperación de acceso mediante correo electrónico o teléfono.

- **Personalización**:
  - Personalización flexible de la apariencia y el comportamiento del widget.

- **Integración**:
  - Integración sencilla con aplicaciones web.
  - Funciona a través de una API REST para la interacción con el servidor.

## Tecnologías

- **React**: El framework principal para construir la interfaz de usuario.
- **TypeScript**: Usado para tipado y mejorar la fiabilidad del código.
- **Redux Toolkit**: Gestión del estado de la aplicación.
- **Material-UI**: Biblioteca de componentes para crear una interfaz moderna.
