# Widget for SSO Encvoy ID

This project is a widget for integrating with the Encvoy ID single sign-on (SSO) system. The widget provides an
interface for user authentication, registration, password recovery, and other access-related features.

## Key Features

- **User authentication**:
  - Sign-in using email, phone, or external providers (OAuth2, etc.).
  - Support for two-factor authentication (2FA).
  - Support for webAuthn and mTLS protocols.

- **Registration and password recovery**:
  - Register new users.
  - Recover access via email or phone.

- **Customization**:
  - Flexible customization of the widget's appearance and behavior.

- **Integration**:
  - Easy integration with web applications.
  - Works via REST API for server-side interaction.

## Technologies

- **React**: The primary framework for building the user interface.
- **TypeScript**: Used for typing and improving code reliability.
- **Redux Toolkit**: Application state management.
- **Material-UI**: Component library for creating a modern interface.
